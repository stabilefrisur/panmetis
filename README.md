# panmetis

A curated collection of agent skills for AI coding assistants.

## Installation

```bash
uv add panmetis
```

Or with pip:

```bash
pip install panmetis
```

## Usage

```python
import panmetis
```

## Development

Clone the repository and sync dependencies:

```bash
git clone https://github.com/stabilefrisur/panmetis.git
cd panmetis
uv sync
```

Build:

```bash
uv build
```
